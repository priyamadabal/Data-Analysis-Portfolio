ML Project: Cervical Cancer Risk Prediction

Project Overview

This machine learning project aims to predict cervical cancer risk by analyzing a dataset of 835 patients using demographic, behavioral, and medical test features. The ultimate goal is to support early diagnosis and assist healthcare providers with data-driven insights. By identifying patterns and risk factors, the project contributes to improving public health outcomes through timely intervention.

Objectives

  • Phase 1 ‚Äì Data Exploration: Focused on cleaning and preparing the dataset by removing irrelevant variables, handling missing values, and performing exploratory data analysis (EDA). Key predictors such as smoking, contraceptive use, and sexual history were identified to guide modeling.
  • Phase 2 ‚Äì Model Development: Applied four machine learning algorithms ‚Äî KNN, Naive Bayes, Decision Tree, and Neural Network ‚Äî to predict biopsy outcomes. Each model was tuned using Grid Search and cross-validation, with Naive Bayes achieving the best AUC score of 0.93.

Methodology

Phase 1: Data Preparation & Exploration

    The dataset ( phase1.csv) from Kaggle, containing 835 patient records and 36 features, was thoroughly inspected. Missing values and low-variance or irrelevant features were removed, and outliers were handled to improve data quality. Exploratory Data Analysis (EDA) helped uncover correlations and highlighted key risk factors like smoking and contraceptive use. Categorical variables were encoded, and numerical features were scaled using MinMaxScaler to prepare the data for modeling.

Phase 2: Model Building & Evaluation

    Using the cleaned dataset ( phase2.csv), four classification models were tested: K-Nearest Neighbors, Naive Bayes, Decision Tree, and Neural Network. Feature selection was refined using Random Forest importance. Each model was built using preprocessing pipelines and fine-tuned with Stratified K-Fold Cross-Validation and Grid Search. Performance was assessed using metrics such as accuracy, precision, recall, F1 score, and AUC-ROC, with the Naive Bayes model delivering the best result, achieving an AUC of 0.930.

Key Findings

Phase 1: Data Insights

  • Smoking, hormonal contraceptive use, and sexual history were identified as strong risk indicators for cervical cancer.
  • Visualizations revealed higher prevalence of risk factors among specific demographic groups.
  • Feature correlation analysis helped reduce dimensionality and enhanced model readiness.

Phase 2: Model Insights

  • Jupyter Notebook: Download Model Notebook
  • Naive Bayes classifier outperformed other models with an AUC score of 0.930, showcasing its effectiveness in binary medical classification.
  • Top three predictive features were Hinselmann test, Schiller test, and Age.
  • Neural Networks performed well but were less interpretable and slightly less efficient compared to Naive Bayes.
  • Decision Trees offered transparency but were prone to overfitting without proper tuning.

Results

  • Naive Bayes had the best performance with ROC AUC = 0.930
  • Top 3 predictive features: Hinselmann test, Schiller test, and Age
  • Evaluation Metrics: Accuracy, Precision, Recall, F1 Score, ROC Curve

Conclusion

The model successfully identified high-risk cervical cancer cases, enabling early intervention. This project shows how machine learning can assist healthcare providers by turning patient data into actionable insights, especially when using interpretable models like Naive Bayes.
